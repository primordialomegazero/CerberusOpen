## What is Cerberus?

Cerberus is a three-headed encryption framework that combines:

| Head | Name | Function |
|---|---|---|
| 🐺 Head 1 | **FHE** | Fully Homomorphic Encryption — compute on encrypted data |
| 🐺 Head 2 | **PQC** | Post-Quantum Cryptography — quantum-resistant security |
| 🐺 Head 3 | **FRACTAL** | FractalTable — unlimited recursive storage |

**One beast. Three heads. Zero compromises.**

docker run -it ghcr.io/primordialomegazero/cerberus-fhe:latest

```
